# MiniPOS — Point of Sale System

> Part of **Systems Universe** | Cost: $0.00/month | Built with Claude Code AI

A complete Point-of-Sale system for small businesses. Runs on Google Apps Script + Google Sheets — zero hosting cost.

## Modules

1. Dashboard — KPI overview
2. POS — Point of Sale with cart & checkout
3. Orders — Order history & details
4. Products — Product catalog CRUD
5. Stock In — Inventory intake
6. Expenses — Cost tracking
7. Customers — Simple CRM
8. Reports — Chart.js visualizations
9. Settings — Store configuration

## Tech Stack

- Frontend: HTML5 + Tailwind CSS + Chart.js + Material Symbols
- Backend: Google Apps Script (V8 JavaScript)
- Database: Google Sheets
- AI Dev: Claude Code
- Design: Google Stitch
